Tecnologias utilizadas

Para crear la página web utilicé HTML,CSS y BOOTSTRAP

Con HTML construí la base y la estructura de la página, colocando los títulos, textos, imágenes y enlaces.

Con CSS le di estilo y diseño, definiendo los colores, las letras, el tamaño de los elementos y cómo se organizan en la pantalla.

Con Bootstrap se implementó como framework de diseño responsivo, facilitando la creación de un sitio adaptable a distintos dispositivos y resoluciones. Gracias a sus componentes predefinidos (botones, menús, formularios, tarjetas, entre otros), se aceleró el proceso de desarrollo y se garantizó una experiencia de usuario consistente.

Gracias a estas dos tecnologías, logré que la página no solo mostrara la información de forma clara, sino que también se viera atractiva y fácil de usar.
